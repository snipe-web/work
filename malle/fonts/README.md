# Шрифты проекта

## Helvetica

Для использования шрифта Helvetica в проекте, пожалуйста, добавьте следующие файлы в эту директорию:

- `Helvetica.woff2` - формат WOFF2 для современных браузеров
- `Helvetica.woff` - формат WOFF для широкой совместимости
- `Helvetica.ttf` - формат TrueType
- `Helvetica-Bold.woff2` - жирное начертание в формате WOFF2
- `Helvetica-Bold.woff` - жирное начертание в формате WOFF
- `Helvetica-Bold.ttf` - жирное начертание в формате TrueType

**Важно:** Убедитесь, что у вас есть соответствующая лицензия на использование шрифта Helvetica в веб-проекте. Шрифт Helvetica является proprietarilyм продуктом Monotype Imaging и требует лицензирования для коммерческого использования.

После добавления файлов раскомментируйте соответствующие строки в файле style.css, чтобы активировать подключение локальных файлов шрифта.
